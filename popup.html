About

Features

Context Dictionary is a powerful browser extension that helps you understand text in any language. Simply select text to look up its meaning or enable real-time translation for seamless reading.

  • Look up word definitions with context
  • Get accurate translations based on context
  • Support for multiple languages
  • Provides related learning resources
  • Clean and intuitive popup interface

How to Use

1. Configure API settings in the popup

2. Select any text on a webpage

3. Definition will appear automatically

4. Or right-click selected text and choose "Get definition"

Keyboard Shortcuts

Alt + D Look up selected text
Alt + T Toggle real-time translation
Toggle the search popup
